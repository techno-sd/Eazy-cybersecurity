# دليل بيانات الاستشارات النموذجية

هذا الدليل يشرح كيفية استخدام البيانات النموذجية للاستشارات في لوحة التحكم.

## الملفات

- **[database/sample-consultations.sql](database/sample-consultations.sql)** - ملف SQL يحتوي على 12 استشارة نموذجية
- **[insert-sample-consultations.js](insert-sample-consultations.js)** - سكريبت Node.js لإدخال البيانات النموذجية

## التثبيت والاستخدام

### الطريقة الأولى: استخدام Node.js (موصى بها)

```bash

# تشغيل السكريبت
node insert-sample-consultations.js
```

### الطريقة الثانية: استخدام MySQL مباشرة

```bash

mysql -h mysql-31dd36a-eazy-db.i.aivencloud.com -P 14507 -u avnadmin -p eazyDb

```

## حذف البيانات النموذجية

لحذف البيانات النموذجية:

```sql
-- حذف جميع الاستشارات النموذجية
DELETE FROM consultations WHERE created_at >= DATE_SUB(NOW(), INTERVAL 11 DAY);

-- أو حذف بيانات محددة بالمعرف
DELETE FROM consultations WHERE id BETWEEN 1 AND 12;

-- أو حذف الجدول بالكامل (استخدم بحذر)
TRUNCATE TABLE consultations;
```

## ملاحظات مهمة

- البيانات تستخدم `DATE_SUB(NOW(), ...)` لإنشاء توقيتات واقعية
- بعض الاستشارات موضوعة كـ "قيد التنفيذ" أو "مكتملة" لعرض مراحل مختلفة
- أرقام الهواتف والبريد الإلكتروني وهمية للتوضيح فقط
- أسماء الشركات عامة ولا تشير إلى شركات حقيقية

## الدعم الفني

في حالة مواجهة مشاكل:
1. تأكد من وجود جدول `consultations` (قم بتشغيل `migrate-admin-tables.js` أولاً)
2. تحقق من إعدادات الاتصال بقاعدة البيانات
3. تأكد من صلاحيات الإدخال في قاعدة البيانات
4. تحقق من ترميز الأحرف (يجب أن يكون utf8mb4)
